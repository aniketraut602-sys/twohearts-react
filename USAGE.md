# How to Use TwoHearts

TwoHearts is a secure dating application with advanced privacy features.

## Accessing the Application
The application is deployed at: **[http://80.225.194.184](http://80.225.194.184)**

## Features

### 1. User Profiles
- **Browse**: View other users' profiles.
- **Connect**: Send connection requests to users you are interested in.
- **Edit Profile**: Update your own bio, interests, and preferences.

### 2. Chat
- **Real-time Messaging**: Chat instantly with your connections.
- **Typing Indicators**: See when the other person is typing.
- **Read Receipts**: Know when your messages have been read.
- **Online Status**: See if your connection is currently online.

### 3. Privacy & Security (New!)
- **Ephemeral Messages**: Messages containing sensitive contact information (like phone numbers) will automatically disappear after 5 seconds to protect your privacy.
- **Contact Detection**: The system warns users who repeatedly share contact information off-platform, helping to keep interactions safe.

## Getting Started
1.  **Sign Up/Login**: Create an account or log in.
2.  **Complete Profile**: Add details to your profile to help others find you.
3.  **Find Connections**: Browse the user list and send requests.
4.  **Start Chatting**: Once a request is accepted, you can start chatting!

## Troubleshooting
- If chat messages don't appear, try refreshing the page.
- Ensure you have a stable internet connection.
